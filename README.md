# python_game_helper
 a game helper by python

作为一名咸鱼玩家，本人日日划水，经常忘清体力，所以酝酿了写脚本辅助的想法。
话说写脚本不是更花时间吗啊喂。
额，应该说我敲代码的兴趣已经超过了原神本身，
既然是兴趣，那么自然不会用按键精灵这种无趣古董，
然后又为了节省时间选择了python，虽然我完全是python新手，并且觉得python混乱、运行低效，但不得不在开发效率上说句真香。
我选择了最好清体力的风本，配队选择温迪、砂糖、阿贝多、雷电将军，
温迪砂糖凑双风，雷神挂元素同时被动提高大招伤害，阿贝多补输出。
经过几天终于探索出了完美的刷本流程，程序方面使用wxpython做UI，win32api和pyautoGUI做模拟输入，最后pyinstaller打包
由于要去控制角色捡圣遗物，又不得不用计算机视觉的思路弄了个寻路算法，
最后能做出来还是很开心的。
不过在展示之前我要先放个声明：1.程序本质是利用windows操作系统api模拟键盘鼠标输入，不会对游戏程序进行任何破坏；
2.极其依赖配队，只能刷风本而且效率普通，使用范围和作用有限，不会破坏游戏环境；
3.个人兴趣，已经开源不会盈利，本人坚决维护计算机信息系统安全。
出于一些考虑隐去了敏感信息
